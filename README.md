# Desafio-Projeto-DIO-e-Santander-Excel-IA-Agregador-IR-Gabriela-Araujo

Projetos desenvolvidos para aprovação no curso: **Santander - Excel com Inteligência Artificial - 2º Semestre**

## Descrição

Este repositório contém a entrega do desafio proposto no curso, com um arquivo de Excel totalmente construído para funcionar como **agregador de dados para declaração de Imposto de Renda (IR)**.

A ferramenta foi criada para ajudar o usuário a **organizar e reunir informações essenciais**, permitindo o controle eficiente das entradas com **validações automáticas**, **menus de navegação** e **funcionalidades extras** para facilitar o uso no dia a dia.

## Arquivos

* `Projeto 2 - DIO e Santander - Excel IA - Gabriela Araujo`: arquivo principal do projeto com o agregador de dados e recursos implementados.

## Objetivo

Demonstrar a aplicação prática dos conceitos aprendidos nas videoaulas por meio da construção de uma solução no Excel que:

* organize e valide dados relevantes para o IR;
* ofereça uma interface amigável com navegação;
* inclua funcionalidades adicionais (ex.: links rápidos e automatizações).

## Funcionalidades (resumo)

* Menus e navegação entre abas
* Validações automáticas para entrada de dados
* Organização estruturada das informações para declaração
* Recursos extras para agilizar o preenchimento (ex.: links rápidos)
